features:
  - title: Unified Search
    details: Query 500+ museums, archives, libraries, and open image sources simultaneously. Results stream in as each source responds.
  - title: Exact + Explore Modes
    details: Exact mode matches your terms precisely. Explore mode casts a wider net across all sources for visual discovery.
  - title: Board View
    details: Pin images to a freeform board. Arrange, group, and compare results from different sources side by side.
  - title: Deep Zoom
    details: IIIF and OpenSeadragon integration for examining high-resolution images down to brushstroke level.
  - title: Colour Extraction
    details: Auto-extract dominant colours from any result. Filter and sort by palette across the entire result set.
  - title: 3D Constellation
    details: Visualise search results as a navigable three-dimensional point cloud based on visual similarity.
  - title: AI Research Assistant
    details: Optional integration with Gemini, Claude, OpenAI, or Ollama. Bring your own key. Works fully without AI.
  - title: Cross-Reference
    details: "Interpret: select any image and cross-reference it against all sources to find related works, provenance, and context."
  - title: Keyboard-Driven
    details: Full keyboard navigation. No mouse required. Designed for fast, fluid visual research workflows.
  - title: Source Filtering
    details: Toggle individual sources or entire categories. 8 categories including museums, photography, nature, historical, art and design.
  - title: Offline Caching
    details: Previously loaded images and metadata are cached locally. Continue browsing your recent searches without a connection.
  - title: Open Source
    details: AGPL-3.0 licensed. Add sources via JSON manifest and PR. Community-driven, no vendor lock-in.
