# Sources

InspoSearch aggregates **500+ sources** across eight categories. Each source is queried in parallel and results stream into the unified results grid.

## Overview

| Category | Sources | Description |
|----------|---------|-------------|
| Museums | 212 | Major and regional museums worldwide |
| Historical | 75 | Archives, national libraries, historical societies |
| Art and Design | 58 | Design collections, illustration archives, typography |
| Photography | 22 | Photo agencies, press archives, street photography |
| Nature | 20 | Natural history, botanical illustration, wildlife |
| Maps | varies | Cartographic collections, historical atlases |
| Fashion | varies | Costume archives, fashion photography, textile design |
| Science | varies | Scientific illustration, medical archives, astronomy |
