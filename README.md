# InspoSearch Docs

Documentation site for [InspoSearch](https://insposearch.pages.dev) — an open-source multi-source visual research engine. 500+ cultural heritage sources, 3B+ images.

**Live docs** → [insposearch-docs.pages.dev](https://insposearch-docs.pages.dev)

**App** → [insposearch.pages.dev](https://insposearch.pages.dev)

## Stack

- [VitePress](https://vitepress.dev) — static site generator
- Cloudflare Pages — hosting
- DM Mono + Cormorant Garamond — typography

## Development

```bash
npm install
npm run docs:dev      # local dev server
npm run docs:build    # production build
npm run docs:preview  # preview production build
```
